# Python Quiz App

Dette er et enkelt Python-program som lar brukeren svare på spørsmål i en quiz.

## Funksjoner

* Flere spørsmål
* Brukerinput
* Poengsystem

## Hvordan kjøre programmet

1. Installer Python
2. Last ned prosjektet
3. Åpne terminal i mappen
4. Kjør:

```bash
python main.py
```

## Hva jeg har lært

* Bruke lister og dictionaries
* Ta input fra bruker
* Lage en enkel logikk med if-setninger
